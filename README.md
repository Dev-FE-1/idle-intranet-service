# (mbt)-idle Intranet Service

이 프로젝트는 토이 프로젝트 1 을 위한 템플릿 프로젝트입니다.
템플릿에는 vite 기반 프론트앤드 프로젝트 구조와 express 기반 node 서버를 포함하고 있습니다.

## 설치

```
npm install
```

## 실행

### 서버 실행

```
npm run server
```

### 클라이언트 실행

```
npm run dev
```

### 커밋 메시지 템플릿 설정

```
git config commit.template .gitmessage.txt
```
